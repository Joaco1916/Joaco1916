Software Engineer and web developer
-----------------

  I am a Software Engineer from Argentina 🇦🇷, currently graduated from the Universidad Nacional del Sur in the city of Bahia Blanca (Buenos Aires, Argentina) where I currently live. I am working in this same city for a company called Razortech dedicated to make software at national and international level as a full-Stack developer.\
\
  My last project was my final project which was about reinforcement learning and 3D simulation of environments, which is already finished and allowed me to graduate. It is an agent 🤖 that learns how to collect and classify garbage from a virtual environment. After this last project I decided to start the path of machine learning, especially focused on reinforcement learning. Although it is a long way I find it entertaining and challenging.\
\
  In addition to programming, I am especially attracted to visual arts and projects. It can be anything from hand drawn drawings to documentaries on how a certain 3D animated character was made for a movie. My favorite hobbies are drawing 📝, jigsaw puzzles 🧩 and swimming 🏊. Piano is also on that list but I'm still learning.
